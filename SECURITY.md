# Security Policy

## Supported Versions

Security fixes are applied to the latest `main` branch.

## Reporting a Vulnerability

Do not open public issues for security reports.

Use GitHub Security Advisories:

- https://github.com/dungle-scrubs/nimoy/security/advisories/new

Include:

- clear reproduction steps
- impact assessment
- affected files and versions
- proof-of-concept if available

## Response Expectations

- Initial triage target: within 3 business days
- Status update target: within 7 business days
- Fix timeline depends on severity and complexity

If the report is valid, coordinated disclosure will be used before public
announcement.
